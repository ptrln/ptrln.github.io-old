Game Play

Controls

key 'a' : move left
moves the landspeeder towards the left.

key 'd' : move right
moves the landspeeder towards the right.

mouse left key : attack
the landspeeder will fire a laser towards the location clicked with 100% accuracy. if the key is held down it will continuously fire.
Only movesments towards the left and right are allowed. Players can move the landspeeder around to dodge enemy fire. Enemy units will always keep their aim on the landspeeder as the landspeeder moves around.

Difficulty Modes

key '1' : easy
This is the easiest mode. Most enemy units' HP and attack probability reduced by half.

key '2': normal
This is the default mode. Enemy units' HP and attack probability will be the default.

key '3' : hard
This is a slightly harder mode. Most Enemy units' HP and attack probability will be multiplied by 1.5.

key '4' : insane
This is an impossible to beat mode. Most Enemy units' HP and attack probability will be multiplied by 5.
The game comes with 4 difficulty modes. You can change modes at any time. You will not have to restart the game every time you change modes but you will restart the current level that you are playing.

Shield

key 's' : enable shield
This enables the shield. Pressing 's' while the shield is on will not extend the current sheild's time.
The landspeeder's shield can be enabled to temporarily block enemy fire. It will stay on to shield enemy lasers for a short period of time, and the shield can sustain an unlimited amount of damage within that time. The shield will start to blink as it's about to finish. By default every player is given 3 shields.



Game Over

key 'r' : restart game
Restarts the game from level 1.
There are 3 different game over screens. Depending on when and how you've reached the game over point you will get different screens. Pressing 'r' will always allow you to restart the game from the beginning.
