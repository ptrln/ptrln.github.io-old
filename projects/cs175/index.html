Introduction

My final project is a Star Wars styled shooter game. The player will control an aircraft to battle various enemy units by using the computer's keyboard and mouse. The player can control the craft using the keyboard and can aim and shoot using the mouse. The controls are highly intuitive.

The game is fully featured including Star Wars style word crawl, sound effects, music and so on.
